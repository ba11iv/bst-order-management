# BST Order Management System

A Java implementation of a Binary Search Tree (BST) used to manage customer orders.

## Project Description
This project demonstrates the use of a Binary Search Tree to store and manage order data efficiently, including insertion, searching, deletion, and tree analysis operations.

## Features
- Insert new orders
- Search for specific orders
- Remove orders
- Find minimum and maximum orders
- Check if the tree is balanced
- Calculate total price of all orders
- In-order traversal display

## Data Structure Used
- Binary Search Tree (BST)
- Generic implementation using Java Generics

## Files
- BST.java
- BSTNode.java
- Order.java

Java
